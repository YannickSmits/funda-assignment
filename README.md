# funda-assignment

## Build Setup

### `api`
```bash
# Go to api folder and install dependencies
$ npm install

# Run below command to start the api on localhost with port 3001
$ node app.js
```

### `app`
```bash
# Go to app folder and install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
